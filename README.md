# 📊 Student Performance Analysis Dashboard

## Project Overview

This Power BI dashboard analyzes student academic performance using attendance, study hours, and subject-wise marks.

## Features

- Student Performance Overview
- Subject-wise Analysis
- Attendance Impact Analysis
- Study Hours Analysis
- Class-wise Comparison
- Trend & Correlation Analysis
- Academic Improvement Insights

## Tools Used

- Microsoft Power BI
- Microsoft Excel
- DAX
- Power Query

## Dataset

The dataset contains 100 students with:
- Subject Marks
- Attendance Percentage
- Study Hours
- Class Information

## Dashboard Pages

- Executive Summary
- Subject-wise Performance
- Attendance Analysis
- Study Hours Analysis
- Class Comparison
- Trend & Correlation
- Academic Insights

## Key Insights

- Higher attendance is associated with better academic performance.
- Students studying more than 5 hours generally score higher.
- Class-wise comparisons identify top-performing groups.
- Subject analysis highlights strengths and improvement areas.
